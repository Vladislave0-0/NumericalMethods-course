# Генераторы случайных чисел №2

Требуется для фиксированного генератора случайных чисел и заданного распределения:
  - Проверить гипотезу о том, что выход генератора распределения следует ожидаемому распределению с использованием критерия Колмогорова-Смирнова (KS).
  - Протестировать свойства генератора случайных чисел (случай битовой последовательности) с помощью некоторых тестов:
    - Фурье;
    - Автокорелляция;
    - Тест на промежутки (Gaps Test).
  - В обоих случаях использовать двухуровневое тестирование.


С выполненным заданием можно ознакомиться в `./Task.pdf`.
