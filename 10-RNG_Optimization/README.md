# Оптимизация генераторов случайных чисел и распределений

Реализовать векторную версию `minstd_rand`. Требования:
  - Код на `C`/`C++`, `fp32`
  
  - Не использовать оптимизированные третьесторонние версии генератора
  - Можно использовать компилятор с поддержкой автовекторизации и поддержать длины степени двойки
  - Подтвердить функциональную корректность битовой версии (например, с помощью `С++`)
  - Результаты должны быть продемонстрированы в виде кодов и отчета, демонстрирующего функциональную корректность и производительность `clocks per element` pеализации

Реализовать параллельную версию бенчмарка **["Вычисление числа рі"](../05-RNG_Algos/)** с использованием на выбор `OpenMP`/`TBB`/`C++ threads`/`Pthreads`, векторной версии `fp32` `minstd_rand`. Количество двумерных векторов 100 млн.

С выполненным заданием можно ознакомиться в `./Task.pdf`.
