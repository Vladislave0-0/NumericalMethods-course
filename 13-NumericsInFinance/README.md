# Численное решение задач финансовой математики

Реализовать формулу для вычисления цены `call` опциона Европейского типа методом Монте–Карло
  - r = 0.05, $\sigma$ = 0.1, $S_0$ = 100, K = 100, T = 1
  - 1 000 000 путей
  - Продемонстрировать функциональную корректность

Возможно использовать доступные библиотеки генераторов случайных чисел, математических функций, линейной алгебры или свои реализации алгоритмов

С выполненным заданием можно ознакомиться в `./Task.pdf`.
