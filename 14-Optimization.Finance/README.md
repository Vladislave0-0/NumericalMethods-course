# Оптимизация в задачах финансовой математики

Предложить вычислительную схему для Black–Scholes Monte–Carlo для случая `n` опционов покупателя на базе аналитического решения и/или интегрирования по времени. Каждый опцион определяется своим набором параметров.
  - Имплементация — на С/С++
  - Параметры генерируются случайным образом, количествово траекторий 100 000, количество опционов 100
  - В имплементации предпочтительнее использовать векторные реализации генераторов случайных чисел/распределений. Возможно использование их скалярных версий из доступных библиотек
  - В каждой из опций учесть специфику технологий для параллелизации (Pthreads, C++ threads, OMP, ТВВ...) и определить возможности для вложенного параллелизма
