# Архитектура программных систем

Разработать (на словах, без кода) и описать архитектуру системы, состоящей из следующего функционала:
  - Математические функции, генераторы случайных чисел, матричное умножение, стохастические процессы, финансовые приложения
  - Результат — короткий отчет, содержащий структуру системы (например, с использованием `UML` или другого языка описания)
  - Для одного класса читателей (разработчиков, QA, менеджеров проекта, архитекторов)
  - Содержащий краткие пояснения о причинах выбранных архитектурных решений

С выполненным заданием можно ознакомиться в `./Task.pdf`.
