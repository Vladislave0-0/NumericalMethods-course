# Оптимизация для современных архитектур

Оптимизировать матричное умножение $С = А \times В$ с размерами матриц, не превосходящих размер `L1`, используемой для оптимизаций архитектуры. Требования:
  - Код на `C`/`C++`, `fp32`
  
  - Использовать матрицы с размерами степеней двойки. Иметь возможность менять размеры. Например: $А^{32 \times 64}$, $В^{64 \times 16}$. Матрицы заполнены генератором случайных чисел
  - В отчете указать вычислительное окружение (архитектура, параметры, компилятор, строка компиляции и т.д.)
  - Необходимо продемонстрировать функциональную корректность реализации
  - Возможно использовать оптимизирующий компилятор с поддержкой автовекторизации
  - `BLAS` библиотеки не вызываются в коде
  - Время вычислений измеряется в тактах. Дать оценку эффективности реализации по сравнению с теоретической для используемых архитектуры и алгоритма

С выполненным заданием можно ознакомиться в `./Task.pdf`.
