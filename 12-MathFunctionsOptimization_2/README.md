# Оптимизация математических функций №2

Ускорить код [ранее написанного](/07-MathFunctions/logf_scalar.c) `logf`, исходя из следующих предположений:

  - Целевая метрика — `throughput`, то есть, имеет смысл векторизовать код.
  - Область определения функции не меняется, работаем только с нормализованными числами.
  - Целевая архитектура — `x86_64` с `AVX2`.
  - Логарифм должен удовлетворять требованиям к точности.

Оформить работу в виде финального отчета с кратким описанием использованного алгоритма, оптимизаций, указанием максимальной ошибки и финальной производительности в `СРЕ`.

С выполненным заданием можно ознакомиться в `./Task.pdf`. Необходимый код:

  - Реализация `logf_avx2` — [07-MathFunctions/logf_avx2.c](/07-MathFunctions/logf_avx2.c)
  - Тестирование точности — [08-MathFunctionsTesting/src/test_vector.c](/08-MathFunctionsTesting/src/test_vector.c)
  - Сравнение производительности — [11-MathFunctionsOptimization_1/src/main.cpp](/11-MathFunctionsOptimization_1/src/main.cpp)
